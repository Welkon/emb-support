# emb-support adapters

Shared chip-support catalog assets live here.

This directory now carries the migrated adapter payload that previously lived in the standalone `emb-agent-adapters` repository:

- `chip-support/`
  Reusable tool routes, algorithms, and shared helpers.
- `extensions/`
  Family, device, and chip profiles used by the runtime.
- `docs/sources/`
  Lightweight reusable source summaries for adapter maintenance.

Use this tree as the source root for:

- `support bootstrap`
- `support sync`
- `adapter export`
- `adapter publish`
