# emb-support skills

Installable skill bundles for emb-agent live here.

The installer can preview this directory and let the user choose which skills to install during `emb-agent` setup.

Current bundle contents:

- `xc8-build`
- `tektronix-scope`

This directory ships an explicit `.emb-agent-plugin/plugin.json` manifest so skill discovery does not depend on legacy path inference.
